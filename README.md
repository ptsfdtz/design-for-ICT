# my-design-for-ICT

This project is a drawing and model file for the competition

print-models 文件夹里是需要打印的模型

models 文件夹里是整个车的模型和装配体，可以使用 ug 或者 soildworks 打开。

images 文件夹里是 PDF 和 CAD 图纸，后面的记号是版本的更迭。
